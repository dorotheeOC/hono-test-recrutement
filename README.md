# Exercice 01

Intégration HTML d'une maquette avec SASS

## Installation

```bash
$ npm install -g yarn
$ yarn set version berry
```

### Dossier ./src

```bash
$ yarn
$ npm install gulp-sass
$ npm install node-sass
$ gulp watch
```
